HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head and body are semantic elements
  • Declarations are not case sensitive but elements are
  • Only use the h1 heading once
  • p element is for paragraphs
  • li element is for lists
  • ul element is for unordered lists
  • img element is for images, which will include src, alt, width, and height attributes
  • br element is for line breaks
  • a element is for links with href attribute for the link itself followed by a description before closing the /a
  • section elements can have id and class attributes
  • div and span are non-semantic elements used to create a division and create an inline division respectively
